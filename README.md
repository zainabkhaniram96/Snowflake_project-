Project Overview :-
This project presents a Car Sales Dashboard built using Power BI to analyze sales performance, customer behavior, and vehicle trends.
The dashboard provides actionable insights through interactive visualizations and key performance indicators (KPIs).

Dataset Description:-
The dataset contains detailed information about car sales transactions, including:
 ~Customer details
 ~Car brand,model and emgine information
 ~Sales transactions
 ~Pricing and revenue data
This data is used to analyze sales trends, customer preferences, and overall business performance.

Key KPIs:-
Total Sales Revenue
Total Cars Sold
Average Selling Price
Top Selling Car Model
Avg Price diffrence
Monthly Sales Trend
YOY Avg Price Growth
YOY Car Sold growth

Dashboard Features:-
Interactive filters (slicers)
 ~Body Style
 ~Dealer Region
 ~Transmission etc
Sales trend analysis (monthly/yearly)
Customer-level insights
Model-wise and brand-wise performance
Region-based sales distribution

Key Insights:-
Certain car models generate higher revenue compared to others
Sales vary significantly across regions
Customer preferences differ based on fuel type and transmission
Sales trends show peak performance during specific months

Tools & Technologies Used
Power BI – Data cleanning, Data modelling,Data visualization
DAX – Calculations & measures
Data source:-xlsx

Project Highlights:-
Data cleaning and transformation performed
Data modeling applied (basic relationships)
DAX measures created for KPI calculations
Interactive and user-friendly dashboard design

How to Use:-
Download the .pbix file
Open in Power BI Desktop
Explore dashboard using filters and visuals
